# Templates for Dradis Framework

This repo contains versioned files that we release as templates for both
Dradis Framework users to use different testing methodologies and show how
different export plugins can be used / customized.

Many of these templates will also be available under Academy > Templates on

https://dradis.com/academy/templates/
